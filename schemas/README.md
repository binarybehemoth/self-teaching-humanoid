# Schemas

Every schema defined across *The Self-Teaching Humanoid* (Appendix O).
Each is a small, versioned contract that lets independent components share one
language — the connective tissue of the whole system, and its audit trail.

| Schema | Carries | In the book |
|--------|---------|-------------|
| `skill_graph_node.json` | A skill and its prerequisites | Parts XI–XIV |
| `replication_authorisation.json` | A human-authorised, bounded replication | Chapter 34 |
| `teaching_contribution.json` | What was taught, by whom, on what | Parts XV–XVII |
| `skill_listing.json` | A skill offered for exchange | Parts XV–XVII |
| `skill_package.json` | A composed skill's dependency tree | Chapter 40 |
| `capability_descriptor.json` | A robot's embodiment, sensors, and safety | Parts XV–XVII |
| `model_exchange.json` | A skill traded robot-to-robot | Parts XV–XVII |
| `commissioning_certificate.json` | Proof a machine's safety gates passed | Chapter 47 |

All are JSON Schema (draft 2020-12).
