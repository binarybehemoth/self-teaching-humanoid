Fleet Dashboard

Human oversight of the fleet — the operator always sees, and can always stop.

This is the human end of the safety architecture. The prominent stop is always reachable; each robot surfaces its safety flag immediately. The monitor on each robot can veto an unsafe action on its own, but a person can always intervene here.

Offline demo: if the back end is not running, sample robots are shown so the surface is legible. Start the API with python3 -m backend.app to see live data.
